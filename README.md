# SQL Practice – Day 1 

## Topics Covered

* Basic SQL Queries
* Data Retrieval using SELECT
* Filtering Data using WHERE
* Pattern Matching with LIKE
* Wildcards (% and _)
* Date-Based Queries
* Aggregate Functions
* GROUP BY Operations

## Concepts Practiced

### Basic Query Operations

* Retrieving all records from a table
* Selecting specific columns
* Applying conditions to filter data

### String and Pattern Matching

* Finding names starting with specific letters
* Finding names ending with certain characters
* Searching names containing particular letters
* Using wildcard characters for pattern searching

### Date Handling

* Filtering records based on year
* Filtering records based on month
* Comparing dates
* Finding recent records using date functions

### Aggregate Functions

* Calculating total salary
* Finding average salary
* Identifying minimum salary
* Counting records

### Grouping Data

* Grouping employees department-wise
* Counting employees in departments
* Finding department-wise average salary

## Learning Outcome

Improved understanding of SQL fundamentals including data retrieval, filtering, pattern matching, date handling, aggregate functions, and grouped analysis. 💡
